# **App Name**: Pure Obscura

## Core Features:

- Void Surface: A clean, absolute black background utilizing Next.js for complete immersion.
- Midnight Whispers: A contemplative AI tool that generates minimalist, one-sentence reflections on stillness when the screen is touched.
- Stealth UI: Auto-hiding controls that only appear on specific gestures to preserve the dark aesthetic.

## Style Guidelines:

- Background: Deep Vanta-black (#050505) to ensure total visual absence.
- Primary Color: Saturated Steel (#666e75) for minimal UI elements.
- Accent Color: Muted Plum (#7d5a7d) to provide a soft, low-light contrast for interactive states.
- 'Inter' sans-serif is recommended for its neutral and machined precision, ensuring high legibility even at small sizes.
- Minimalist, edge-to-edge layout with a focus on negative space.
- Ultra-slow fade-ins (2s duration) for AI text to maintain a peaceful environment.
- Outline icons with a 0.5px stroke to keep the visual weight as low as possible.
